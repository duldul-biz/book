# AA算法

### SSAA 
解决的是几何画线的锯齿  用一个更大的分辨率来渲染场景
![](Img/2018-12-07-22-04-25.png)



### MSAA
解决的是几何画线的锯齿

![](Img/2018-12-07-21-44-50.png)




我们只对每个像素进行着色，但是并不意味着我们只需要存储一个颜色值，而是需要为每一个子采样点都存储颜色值，所以我们需要额外的空间来存储每个子采样点的颜色值。所以，颜色缓冲区的大小也为非MSAA下的n倍。
